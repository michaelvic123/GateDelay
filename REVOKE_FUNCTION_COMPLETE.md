## 🚀 Getting Started

### 1. Review Documentation
```bash
# Start with the README
cat GateDelay/Contracts/REVOKE_FUNCTION_README.md

# Quick start guide
cat GateDelay/Contracts/REVOKE_FUNCTION_QUICK_START.md
```

### 2. Build Contracts
```bash
cd GateDelay/Contracts
forge build
```

### 3. Run Tests
```bash
# Run all RevokeFunction tests
forge test --match-contract RevokeFunctionTest -vv

# Run example tests
forge test --match-contract RevokeFunctionExampleTest -vv

# Run all with gas report
forge test --match-path "test/RevokeFunction*.sol" --gas-report
```

### 4. Deploy
```bash
# Set environment
export PRIVATE_KEY=your_private_key

# Deploy to testnet
forge script script/DeployRevokeFunction.s.sol:DeployRevokeFunction \
  --rpc-url $TESTNET_RPC_URL \
  --broadcast
```
